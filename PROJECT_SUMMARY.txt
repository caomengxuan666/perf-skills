QUICK START
-----------
1. Install perf: sudo apt-get install linux-tools-common linux-tools-$(uname -r)
2. Setup permissions: sudo setcap "cap_perfmon,cap_sys_admin,cap_sys_ptrace+ep" /usr/bin/perf
3. Record: perf record -p <PID> -g -o perf.data -- sleep 30
4. Analyze: perf report -i perf.data -g graph
5. Optimize based on Self% hotspots

For detailed guidance, see QUICKSTART.md
